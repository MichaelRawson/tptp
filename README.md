# tptp

A [crate](crates.io/crates/tptp) for reading files in the [TPTP](tptp.org) format.

## Features

* Streaming parser
* Flexible `include()` handling with the `Resolve` trait
* Reasonably complete FOF support
* Sensible AST
* String sharing
